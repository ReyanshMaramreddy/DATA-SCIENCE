### How to typecast a list into numpy array:
```
import numpy as np
li = [1,2,3]
linp = np.array(li)
print(type(li))
print(type(linp))
```
### Statistical Measures in numpy:
```
arr = np.array([1, 2, 3, 4, 5])
mean = np.mean(arr)
median = np.median(arr)
std_dev = np.std(arr)
max_val = np.max(arr)
min_val = np.min(arr)
print(mean, median, std_dev,max_val, min_val)
```
